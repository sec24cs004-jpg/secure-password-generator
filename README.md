Secure Password Generator

Overview

Secure Password Generator is a Python-based application that generates strong and random passwords. It helps users create secure passwords containing uppercase letters, lowercase letters, digits, and special characters.

Features

- Generate random passwords of any length
- Includes letters, numbers, and special symbols
- Displays password strength
- User-friendly menu-driven interface
- Generates multiple passwords in a single session

Technologies Used

- Python 3
- Random Module
- String Module

How to Run

1. Install Python 3.
2. Download the project files.
3. Open a terminal or command prompt.
4. Run: python password_generator.py. 
5. Enter the desired password length.
6. Copy the generated password.

Sample Output:

Secure Password Generator

Enter password length: 12

Generated Password:
A@9kL#2mP!7x

Password Strength: Strong
